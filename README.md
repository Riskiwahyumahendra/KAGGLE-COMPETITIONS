## ⚙️ Installing and Running the Projects on Windows

Follow these steps to download the repository and set up your environment for running the machine learning notebooks.

### Step 1: Download the Repository

Click the big green badge above or visit the link below to download the repository to your computer:

[https://github.com/Riskiwahyumahendra/KAGGLE-COMPETITIONS](https://github.com/Riskiwahyumahendra/KAGGLE-COMPETITIONS)

You will be taken to the GitHub page where you can download the repository as a ZIP file.

- Click the green **Code** button near the top right of the page.  
- In the dropdown, select **Download ZIP**.  
- Save the ZIP file to a folder you can easily access.

### Step 2: Extract the Files

After downloading:

- Locate the ZIP file you saved (usually in your Downloads folder).  
- Right-click the file and select **Extract All...**  
- Choose a destination folder like `Documents\KaggleProjects` to keep the files organized.  
- Click **Extract** to unpack the files.

### Step 3: Install Python (if not already installed)

If your PC does not have Python installed, follow these steps:

- Go to the official Python site: [https://www.python.org/downloads/](https://www.python.org/downloads/)  
- Download the latest version for Windows (make sure it’s version 3.8 or higher).  
- Run the installer and follow the instructions. Ensure the box **Add Python to PATH** is checked during installation.

### Step 4: Install Required Packages

The projects use common Python packages for data science. To install them:

- Open the **Command Prompt** by pressing `Win + R`, typing `cmd`, and hitting Enter.  
- Navigate to the folder where you extracted the files by typing something like:  
  `cd C:\Users\YourName\Documents\KaggleProjects\KAGGLE-COMPETITIONS-main`  
- Run this command to install all needed packages:  
  `pip install -r requirements.txt`

This command reads a list of required packages and installs them automatically.

### Step 5: Launch Jupyter Notebook

The projects are in Jupyter Notebook format. To open them:

- In the same Command Prompt window, type:  
  `jupyter notebook`  
- A browser window will open showing the folder contents.  
- Click on any notebook file (`*.ipynb`) to open it.

### Step 6: Running the Notebooks

- Each notebook contains cells with code and explanations.  
- Click on a cell and press `Shift + Enter` to run the code in that cell.  
- Follow the instructions and comments inside the notebooks to understand each step.
